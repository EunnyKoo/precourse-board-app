# precourse-board-app
# 개커플 (개발자들을 위한 커뮤니티 플랫폼)

![프로젝트 로고](프로젝트로고.png)

**개커플**은 항해 99 프리코스 1기 토이 프로젝트로 개발된 커뮤니티 플랫폼입니다.   
이 프로젝트는 CRUD(Create, Read, Update, Delete) 작업을 연습하고 구현하기 위한 목적으로 시작되었습니다.

## 주요 기능

- 사용자 등록, 로그인, 로그아웃 기능
- 게시물 생성, 조회, 수정, 삭제 기능

## 스펙

- Java: 17.0.7
- Gradle: 8.2.1
- Spring Boot: 3.1.2
- Lombok 사용
- Spring Web 사용
- H2 Database 사용
